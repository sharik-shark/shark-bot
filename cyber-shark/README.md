**Installation steps** 

```
> pkg install git
> pkg install wget
> pkg install ffmpeg
> pkg install nodejs
> pkg install npm
```
**Clone This**
```bash
> git clone https://github.com/KCG-BOT/ck-wp-bot
> cd KCG-BOT
```
**Continue with**
```bash
> bash install.sh
```
**If you experience an error, perform the following manual commands :**
```bash
> pkg install tesseract
> npm i -g cwebp
> npm i -g ytdl
> npm i node-tesseract-ocr
> npm i
> npm i got
```
**Run the bot**
```bash
> node index.js
```
**Finally, scan the QR code**
